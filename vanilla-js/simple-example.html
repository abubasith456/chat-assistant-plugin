Simple Chat Widget Integration

This is a basic example of how to integrate the chat widget into your website.

Integration Steps:

  1. Include the CSS and JavaScript files
  2. Call ChatWidget.init() with your configuration
  3. The widget will appear automatically!
<link rel="stylesheet" href="chat-widget.css">
<script src="chat-widget.js"></script>
<script>
  ChatWidget.init({
    position: 'bottom-right',
    primaryColor: '#10B981',
    title: 'Customer Support'
  });
</script>

Look at the bottom-right corner - the chat widget should be visible!

You can customize colors, position, size, and content by modifying the configuration object.
