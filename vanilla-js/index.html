Usage Instructions

To use this chat widget in your website:

1. Include the CSS and JS files:

<link rel="stylesheet" href="chat-widget.css"> <script src="chat-widget.js"></script>

2. Initialize the widget:

// Basic usage with default settings ChatWidget.init(); // Custom configuration ChatWidget.init({ position: 'bottom-left', primaryColor: '#10B981', headerBgColor: '#059669', title: 'Support Chat', width: '400px', height: '600px', placeholder: 'How can we help you?' });

3. Multiple widgets (different configurations):

// You can create multiple instances const widget1 = ChatWidget.init({ position: 'bottom-right', title: 'Sales Chat' }); const widget2 = ChatWidget.init({ position: 'bottom-left', title: 'Support Chat', primaryColor: '#10B981' });
